# ФинКлик — независимая платформа учёта и отчётности

## White Label для банков как дополнительная услуга

---

## Проблема

**4 из 5** предпринимателей в Беларуси ведут учёт в Excel или на бумаге.

| Боль клиента банка | Последствия для банка |
|---|---|
| Нет автоматизации учёта | Клиент уходит к конкурентам с экосистемой |
| Ошибки в отчётности → штрафы | Закрытие бизнеса = потеря клиента |
| Нет интеграции «банк ↔ учёт» | Низкая вовлечённость в ДБО |
| Дорогой бухгалтер | Клиент экономит, не растёт → меньше оборотов |

**Банки с экосистемой** (расчётный счёт + учёт + отчётность) удерживают **на 40% больше МСБ-клиентов** и увеличивают ARPU в 2-3 раза.

---

## Решение: независимая платформа ФинКлик + White Label для банков

**ФинКлик** — самостоятельная платформа для учёта, налогов и отчётности малого бизнеса.

Для банков мы предоставляем **дополнительную услугу White Label**: запуск под вашим брендом за **4-8 недель** вместо 12-18 месяцев разработки с нуля.

## Ключевые модули

### 1. Финансовый учёт
- Операции: доходы, расходы, возвраты, списания
- 10 категорий расходов (зарплата, аренда, материалы, маркетинг, налоги...)
- Импорт из CSV / банковской выписки
- Экспорт в PDF, CSV, TXT

### 2. Интеграция с банком (ваше конкурентное преимущество)
- Автоматическая загрузка выписок через API банка
- Баланс счетов в реальном времени
- Создание платежей из интерфейса учёта
- Мульти-банк: поддержка нескольких счетов

### 3. Налоги и отчётность
- Расчёт УСН (3%/5%), НДС, ФСЗН, Белгосстрах
- Формирование деклараций (НДС, ПУ-3)
- Календарь дедлайнов с push-уведомлениями
- Автоматическая подача отчётов (ФСЗН, ИМНС, Белгосстрах, Белстат)

### 4. AI-модуль
- OCR сканирование чеков, ТТН, актов, счетов-фактур
- Извлечение реквизитов, сумм, НДС, контрагентов
- ИИ-консультант по вопросам учёта
- Точность распознавания ТТН: >90%

### 5. Кадры и зарплата
- Реестр сотрудников с шифрованием ПДн (AES-256)
- Расчёт зарплаты: оклад → подоходный → ФСЗН → на руки
- Зарплатные ведомости с экспортом

### 6. Мониторинг законодательства
- Автоматическое отслеживание изменений (ИМНС, ФСЗН, Белгосстрах, Белстат)
- Уведомления о новых формах, ставках, сроках
- Фильтрация по органам и категориям

## Безопасность (критично для банка)

| Требование | Реализация |
|---|---|
| Шифрование ПДн | AES-256 (Fernet), все персональные данные зашифрованы at rest |
| Аутентификация | JWT с refresh-токенами, 15 мин. жизнь access-токена |
| Brute-force защита | Блокировка после 5 неудач на 15 минут |
| Rate limiting | 120 req/min на пользователя, burst до 30 |
| Security headers | CSP, X-Frame-Options DENY, HSTS, XSS protection |
| Аудит-лог | Все действия: кто, что, когда, IP |
| Мульти-тенантность | Изоляция данных по organization_id на уровне каждого запроса |
| HTTPS | TLS 1.2+ обязателен для production |

## Масштабирование

Платформа спроектирована для **40 000+ клиентов**:

| Нагрузка | Инфраструктура | Стоимость |
|---|---|---|
| До 100 клиентов | 1 VPS (4 CPU, 8 GB) | ~50 USD/мес |
| 100-500 | 2-3 реплики API + Managed PostgreSQL | ~300 USD/мес |
| 500-5000 | Kubernetes, HPA, read-replicas | ~800 USD/мес |
| 5000-40000 | K8s cluster, шардирование, CDN | ~2500 USD/мес |

**Индексы БД** оптимизированы: по organization_id, дате, категории, статусу.
**Connection pooling**: 20 + 30 overflow с pre-ping.
**Redis кэш**: дашборд, баланс, налоговый календарь.

## Что входит в услугу White Label

### Кастомизация под бренд банка
- Логотип, цвета, шрифты, favicon
- Домен банка (например, `business.yourbank.by`)
- Email-уведомления от имени банка
- Текст дисклеймеров и пользовательских соглашений

### Интеграция с АБС
- API для автоматической загрузки выписок
- Webhook-уведомления о новых операциях
- SSO (Single Sign-On) через систему аутентификации банка
- Обмен данными по контрагентам

### Поддержка и развитие
- SLA на отклик / устранение инцидентов
- Ежемесячные обновления (новые формы, ставки, законы)
- Выделенный канал поддержки
- Обучение сотрудников банка

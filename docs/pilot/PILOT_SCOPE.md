# FinKlik — Scope пилота (для клиента)

**Версия:** 1.0 · **2026-05**  
**Аудитория:** владелец бизнеса, бухгалтер, команда внедрения

FinKlik — **операционная система для денег и отчётности** малого бизнеса в Беларуси. Пилот — это **закрытая beta**: вы работаете с реальными документами и процессами, а мы дорабатываем продукт по вашей обратной связи.

---

## Что работает в пилоте

| Область | Что вы получаете |
|---------|------------------|
| **Учёт** | Журнал операций, категории, контрагенты, черновики и проводки |
| **Документы** | Загрузка чеков и первички, распознавание (OCR) с **обязательной проверкой** человеком |
| **Банк** | Счета, импорт выписки, сверка (ручной импорт или тестовый контур) |
| **Налоги** | Расчёт по вашему режиму, календарь сроков, подсказки по допущениям |
| **Отчётность** | Пошаговая подготовка и **репетиция подачи** (workflow), экспорт файлов |
| **Задачи** | Входящие, согласования, лента работы, планёр |
| **Команда** | Приглашение бухгалтера / менеджера, роли доступа |
| **Бухгалтер (firm)** | Несколько клиентов, очереди OCR и согласований по всем организациям |

---

## Что пока **не** входит в пилот

| Область | Статус | Что это значит для вас |
|---------|--------|------------------------|
| **Подача в ИМНС / ФСЗН «в один клик»** | Репетиция | Вы проходите процесс и получаете файлы; **юридически значимая подача** — через портал органа или с вашим бухгалтером |
| **ЭЦП и официальные XML** | В разработке | Подпись и форматы порталов — после договоров с органами |
| **Автосинхронизация с банком** | По запросу | Импорт выписки вручную или через тестовый OAuth; live-партнёрство с банком — отдельный этап |
| **Оплата подписки (ЕРИП / карта)** | Не в пилоте | Доступ по приглашению |
| **Общая система налогообложения (ОСН)** | Не в MVP | Пилот: ИП (единый налог), УСН, упрощённый контур |
| **1С в production** | По настройке | Интеграция возможна при согласованном endpoint; иначе — ручной обмен |

---

## Как мы работаем в пилоте

1. **Invite-only** — доступ по приглашению, онбординг профиля бизнеса (~5 мин).
2. **OCR с проверкой** — система предлагает поля, вы подтверждаете; сложные документы могут требовать ручного ввода.
3. **Один главный экран** — деньги, риск, следующий шаг, готовность отчётности.
4. **Обратная связь** — канал с командой FinKlik (Telegram / email); приоритет — блокеры отчётности и OCR.

## Честные формулировки (можно переслать клиенту)

> FinKlik в пилоте помогает **вести деньги, документы и подготовку отчётности в одном потоке**.  
> Это **не замена** личного кабинета МНС/ФСЗН и **не гарантия** автоматической legal-подачи.  
> Ценность пилота — **скорость решений**: сколько денег, что срочно, что мешает сдать отчёт.
